# Bloom

Playable worlds for guitarists.

Bloom is a Windows-first JUCE standalone application and VST3 plugin built around Neural Amp Modeler. The first milestone is **Radiance**: one curated world, one authorised `.nam` model, persistent calibration, and a minimal interface that gets out of the guitarist's way.

## Current milestone

- Repository scaffold
- Windows CI
- JUCE standalone + VST3 target
- Local `.nam` loading
- Persistent input/output calibration
- NeuralAmpModelerCore adapter

## Build status

The repository is being brought up through GitHub Actions on Windows. Until CI passes, treat the code as an active prototype rather than a release build.
